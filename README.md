## Overview

It covers five core tasks: creating an awareness infographic, building a digital portfolio, exploring online platforms, practicing professional communication, and studying cybercrime.

## Tasks Completed

### Task 1 – Digital Literacy Infographic
- Designed an infographic using **Canva** covering digital literacy concepts, tools, safe internet practices, and professional online presence.

### Task 2 – Digital Portfolio
- Created and updated profiles on:
  - **GitHub** – to showcase coding projects
  - **LinkedIn** – to build a professional network
  - **Kaggle** – for data science learning and projects

### Task 3 – Coding Platforms & Collaboration Tools
- Solved a beginner-level problem on **HackerRank**
- Created a 5-question survey using **Google Forms** related to digital literacy

### Task 4 – Professional Communication
- Wrote two professional emails:
  - An assignment extension request
  - An internship application
- Created a social media Do's and Don'ts checklist

### Task 5 – Cybercrime Awareness
- Studied **UPI fraud**, a common cybercrime in India
- Documented how scams work and compiled prevention tips
- Key takeaways: never share OTPs/PINs, verify senders, report fraud at [cybercrime.gov.in](https://cybercrime.gov.in) or call helpline **1930**
